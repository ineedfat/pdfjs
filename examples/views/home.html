PdfJS is built around PDF version 1.3 specification that allows alot of flexibility for on demand client-side PDF generation. It attempts to provide a low level API that closely resembles the PDF specification.

var doc = new pdfJS.doc('letter', 'portrait'); doc.addPage(); //Title doc.currentPage.graphic('pushState'); doc.currentPage.graphic('translate', 160, 700); doc.currentPage.text('beginText'); doc.currentPage.text('fontStyle', null, null, 50); doc.currentPage.text('showText', 'Hello World!'); doc.currentPage.text('fontStyle', null, null, 20); doc.currentPage.text('textPositionWithLeading', 200, -30); doc.currentPage.text('showText', 'pdfJS by iNeedFat :\\)'); doc.currentPage.text('endText'); doc.currentPage.graphic('popState'); //Image var imgXObj = doc.newImage('./img/sample.jpg', true); imgXObj.addImageToPage(doc.currentPage, 60, 50, 300, 100); //More Text doc.currentPage.graphic('translate', 400, 175); doc.currentPage.text('beginText'); doc.currentPage.text('fontStyle', null, null, 15); doc.currentPage.text('showText', 'It\'s just that easy :D!'); doc.currentPage.text('endText');

Note

It is good to understand the internal structure of a PDF file by referencing the 'Adobe PDF 1.3 Reference'.
