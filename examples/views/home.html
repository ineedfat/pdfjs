PdfJS is built around PDF version 1.3 specification that allows alot of flexibility for on demand client-side PDF generation. It attempts to provide a low level API that closely resembles the PDF specification.

var doc = new pdfJS.doc('letter', 'portrait'); var p = doc.addPage(); p.pushState(); //Title p.translate(160, 700); p.beginText(); p.fontSize(50); p.print('Hello World!'); p.endText(); p.beginText(); p.fontSize(20); p.textPosition(200, -30); p.print('pdfJS by iNeedFat :)'); p.endText(); p.popState(); //Image var imgXObj = doc.newImage('./img/sample.jpg', true); p.addImage(imgXObj, 60, 50, 300, 100); //More Text p.translate(400, 175); p.beginText(); p.fontSize(15); p.print('It\'s just that easy :D!'); p.endText();

Note

It is good to understand the internal structure of a PDF file by referencing the 'Adobe PDF 1.3 Reference'.
